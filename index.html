Ninja Chop 2D - Shuriken

"Ninja Chop 2D - Shuriken" is a 2D puzzle and drag-and-drop mechanics game where you play as a ninja chef. Your objective is to slice four fruits per level by throwing shurikens using a drag-and-drop mechanic. Each level features increasing difficulty with unique elements such as portals and disappearing obstacles. You have three attempts to slice all the fruits; otherwise, you fail the level. If a fruit is sliced successfully, a happy emoji appears on the screen; if not, an angry emoji is displayed. The game includes 25 levels, each offering a distinct challenge.
